Recipe Tool is an application that will recommend recipes based on what ingredients you have. It was built for the AP CSP exam by Jasper Mayone and Tristan Gottshall, who are sophomores at Harwood Union High School (class of 2025). The whole project was written from scratch with no outside assistance. Tristan wrote the HTML, CSS, and JavaScript while Jasper was responsible for databasing, color scheme, and website deployment. To see Recipe Tool's source code, go to Recipe Tool on GitHub.
